# Auth-System-GO
Auth System — это полнофункциональная система аутентификации и авторизации, разработанная на языке Go. Система поддерживает два способа верификации пользователей: через SMS-код и электронную почту. Проект реализует современные подходы к безопасности, включая JWT-токены, двухфакторную аутентификацию и защищенное хранение данных.
